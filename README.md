# USB-Security-key.
I want to make this key a type-c key, which i can use to log into my computer or account with for better security. (at the end i had to use USB-A)

So i initially started with making this a usb-c security key, because i would be able to use it on all my devices, however, since the thickness of the board would need to be around 1.6 mm, it would cost too much in jlcpcb, therefore i had to disregard this idea, and move to a usb-type a port. I also wanted this to be assembled by jlcpcb, but since the components are on both sides of the board, it would cost approximately 105 usd to only assemble the board. 

Therefore i came with the idea that i would like to use a high melting solder on the side with least components, and the other side, i want to use solder that will melt in the temperature of around 180 degrees.
